)

Calculates the word vector and stores this information.

Parameters
corpusThe corpus that contains the information on the PubMed article
pmidsA list of PubMed ID tuples to be added to the word vector, where it's the PubMed ID number followed by the qualifier The propagated PubMed IDs will be used if none is given
tokenizerThe tokenizer function that will be used on the text, a simple tokenizer is used if none is given. Should take a string as an input, and outputs a string with words that are lower case and separated by a space
stemmerThe stemmer function that will be used to stem the words, the porter stemmer is used if none is given Takes a word as an input and reports a stemmed word as an output.
stopwordsA list of stop words that will not be included in the word vector, either as a list or a StopwordList. An empty list is used if no stop word list is given.
